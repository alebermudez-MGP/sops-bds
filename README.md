## 📖 Cómo Usar Este Sistema

1. **Busca el tema** en la navegación arriba
2. **Haz clic** en el SOP que necesitas
3. **Lee** el procedimiento en Markdown (fácil de leer)
4. **Usa `Ctrl+F`** para buscar palabras clave dentro del SOP
5. **Sigue los pasos** exactamente como aparecen

### Ejemplo: Necesito verificar la información del cliente
1. Haz clic en [**BDS Portfolio**](bds-operations/bds-portfolio.md)
2. Busca "Account Creation" con `Ctrl+F`
3. Sigue los pasos en la sección "How to Create an Account?"
